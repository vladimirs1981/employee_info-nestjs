# Employee Info Api

## Description

Employee Info Api using [Nest](https://github.com/nestjs/nest) framework.

## Installation

```bash
$ npm install
```

## Before running the app

```bash
# migrate the database
$ npm run db:migrate

# seed the database
$ npm run db: seed
```

## Running the app

```bash
# development
$ npm start

# production mode
$ npm run start:prod
```

## Additional command

```bash
# delete database
$ npm run db:drop

# create migration file after changing entity
$ npm run db:create migration_file_name
```
